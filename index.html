    Read Me

    This project was realized using Bootstrap and Kineticjs.
    You can download the source from github.

    Instructions:

    • To add state click "Add State" button

    • To connect two states : right click on a state and choose "Connect To" menu , then click on the target state.

    • Left click on a state or edge will show "Properties" on the right where you can edit properties of selected item.

    • Every graph must have an intial state ,to set an initial state : right click on a state and select "Set As initial state" menu.

    • After finishing drawing your graph you can start simulation by clicking "Simulation" button then you can insert your symbols for simulation.

    • You can import/export the graph as XML for future use.

    Note: The "&" symbol represents the empty symbol.

    This simulator does not detect ambigus Turing Machine so please check your input before starting simulation to avoid any incoherent result.
